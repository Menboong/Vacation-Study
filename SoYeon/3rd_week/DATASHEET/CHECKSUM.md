# checksum

중복 검사의 한 형태로 오류 정정을 통해 송신된 자료의 무결성을 보호하는 단순한 방법

- checksum 계산 방법
	1. 주어져 있는 모든 바이트를 더한다.
	2.  최상위 4비트를 버린다.
	3. 버린 수의 2의 보수를 얻는다. (1의 보수의 값+1) = checksum 바이트
	4. checksum 바이트를 확인하려면 원래 주어져 있는 바이트에 checksum 바이트를 더한다.
	5. 다시 최상위 4비트를 버렸을 때 0x00이 되면 오류가 없다는 것이다.

## UBlox M8 GPS
1. Buffer[N]에는 checksum을 계산할 데이터가 들어있다.
2. 2개의 checksum값은 오직 8비트 unsigned integer이어야한다.
3. 더 큰 크기의 정수 값으로 실행한다면 loop에서 CK_A와 CK_B 작업이 모두 끝난 후에 두 값을 모두 0xFF로 초기화해야한다.
4. loop 후의 2개의 U1값은 패킷의 끝에서 전송되는 checksum을 포함한다.
