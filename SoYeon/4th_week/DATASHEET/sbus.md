# SBUS

###  한 패킷의 크기
- 25 바이트 크기로 이루어져 있다.
- [시작바이트][data 1][data 2]...[data22][flags][끝바이트]
### 패킷 전송 주기
- analog mode: 14ms 마다
- highspeed mode: 7ms 마다
### 한 바이트의 구성
한 바이트 = 시작비트 1개 + 데이터비트 8개 + 패리티비트 1개 + 정지비트 2개
=> 총 12비트로 구성 

### 기타
- baudrate(보드 속도- 비트/초의 단위를 사용하는 데이터 전송 속도)는 100'000 bit/s
- 하나의 비트는 10μs(마이크로초)의 속도가 걸린다.
- 최상위비트는 가장 먼저 보내진다.
- 시작 바이트: 11110000b (0xF0)
- 끝 바이트: 00000000b
- flag
	- bit 7= ch 17= 디지털 채널(0x80)
	- bit 6= ch 18= 디지털 채널(0x40)
	- bit 5= 프레임 손실(0x20)
	- bit 4= 안전장치 활성화(0x10)
	- bit 3, 2, 1= 해당없음(또는 이용할 수 없음)
- 각 채널 정보는 11비트로 담겨져 전송된다.
	- 따라서 첫번째 바이트에서 8 비트, 그 다음 비트에서 3 비트를 이용한다. (한 바이트 당 데이터 비트는 8비트)
