# 공용체(union)

서로 다른 데이터형을 갖는 변수를 하나의 메모리 공간에 중첩해서 사용하는 것

	union  공용체_태그_이름
	{
        자료형 공용체_항목1 ;

        자료형 공용체_항목2 ;

        ............

        자료형 공용체_항목n ;
	} ;

## 특징
- 공용체의 크기는 공용체 안의 메모리의 가장 큰 변수의 크기를 따라간다.
- 많은 부분에서 구조체와 비슷한 특징을 지닌다.
- 사용될 데이터의 자료형이 유동적인 경우, 공용체를 사용하게 되면 메모리 공간을 효율적으로 사용할 수 있게 된다.
- 여러개의 자료형이 동일한 메모리 공간을 사용한다.


## 사용용도
- 저장하고자 하는 목적은 같은데, 다른 식으로 표현되어야 할 때 사용한다.
- 특정 공간을 효율적으로 관리하기 위해서 사용한다.
- 어떤 경우에는 필요하고 어떤 경우에는 필요하지 않을 때 사용한다.
