checksum
=============
- 체크섬(checksum)이란?
	- 중복검사의 한 형태로, 오류 정정을 통해, 공간이나 시간 속에서 송신된 자료의 무결성을 보호하는 방법을 말한다

- 체크섬 특징
	- 기본적인 메세지 구성 요소 추가하여 결과값 저장함으로써 동작
	- 누구나 데이터에 같은 작업 수행 가능
	- 무결성 검사에 대한 결과 비교 할 수 있음
	- 메시지가 손상되지 않았다는 결론 내릴 수 있음

- 계산 방법
 	- 패킷 전송 시 체크섬은 마지막에 계산 된다 그 때 체크섬의 값은 0으로 초기화 되어야 함
		- 1. 모든 바이트를 더한다
		- 2. 최상위 4비트를 버린다
		- 3. 2의 보수로 바꾼다 (체크섬 바이트)
		- 4. 주어진 값과 2의 보수로 바꾼 값을 더해준다.
		- 5. 다시 최상위 4비트를 버렸을 때 0x00이 된다면 오류가 없다는 것이다.
