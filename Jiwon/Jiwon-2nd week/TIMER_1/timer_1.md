Timer interrupt
=============
- Interrupt 란?
	- CPU가 일을 하고 있는데 거기에 태클을 거는 행위

- timer interrupt란?
	- 특정한 시간이 되면 인터럽트가 발생하게 되는 것
 		- Ex) A라는 작업을 하던 도중 A를 중단(보류)하고 B라는 일을 수행하게 하는 것이 인터럽트
    이 태클을 거는 행위를 타이머 인터럽트, 인터럽트를 걸고 수행시키는 일을 인터럽트 서비
    스 루틴

- timer interrupt의 사용
	- 인터벌 타이머
  		- 태스크 주행시간 감시, 일정시간 경과하면 내부 인터럽트 발생하여 멀티 프로그래밍에서 하나의태스크가 부당하게 오랜 기간 CPU를 독점 하는 것 방지
	- CPU 타이머
	- TOD 타이머
	- 외/내부 장치의 시간 제어
	- 주기적인 인터럽트를 통한 외부/내부 장치 감시 및 제어
	- 정확한 시간을 재기 위해서
